# Lessons Learned

**Version:** v1.0 &nbsp;|&nbsp; **Status:** Active &nbsp;|&nbsp; **Owner:** WES

How WES captures knowledge from completed work. Lessons are recorded at milestones and project closure ([Project Operating System](../../Projects/Project-Operating-System/Project-Reporting.md)) and stored in [`/KnowledgeBase/Lessons-Learned`](../../KnowledgeBase/Lessons-Learned/README.md).

## What to Capture

| Element | Description |
|---------|-------------|
| **Successes** | What worked well and should be repeated. |
| **Failures** | What went wrong and should be avoided. |
| **Risks** | Risks that materialized or emerged. |
| **Improvements** | Concrete changes to process or practice. |
| **Recommendations** | Guidance for future projects. |

## Rules

- Capture lessons briefly and honestly.
- Feed improvements back into standards, templates, and [best practices](./Best-Practices.md).

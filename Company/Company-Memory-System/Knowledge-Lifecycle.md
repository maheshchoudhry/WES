# Knowledge Lifecycle

**Version:** v1.0 &nbsp;|&nbsp; **Status:** Active &nbsp;|&nbsp; **Owner:** WES

The lifecycle of organizational knowledge at WES.

```
Creation → Validation → Approval → Publication → Review → Update → Archive
```

| Stage | Definition |
|-------|------------|
| **Creation** | Knowledge is written down (decision, lesson, practice, or doc). |
| **Validation** | Checked for accuracy and relevance by the responsible role. |
| **Approval** | Approved by the appropriate authority ([Knowledge Governance](./Knowledge-Governance.md)). |
| **Publication** | Published to the Knowledge Base and made discoverable. |
| **Review** | Revisited on a cadence or when circumstances change. |
| **Update** | Revised and re-versioned as needed. |
| **Archive** | Outdated knowledge is archived, not deleted, preserving history. |

Mirrors the information lifecycle in [Volume 09](../../Blueprint/Volume-09-Knowledge-Management/README.md).

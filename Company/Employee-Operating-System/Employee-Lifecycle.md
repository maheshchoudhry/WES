# Employee Lifecycle

**Version:** v1.0 &nbsp;|&nbsp; **Status:** Active &nbsp;|&nbsp; **Owner:** WES

The company-wide lifecycle every AI employee moves through.

```
Recruitment → Activation → Training → Assignment → Review → Upgrade
                                             ↓
                              Temporary Suspension / Retirement
```

| Stage | Definition |
|-------|------------|
| **Recruitment** | A role need is identified and a new employee profile is defined. |
| **Activation** | The profile is completed and the employee is marked Active in the workforce. |
| **Training** | The employee is equipped with the context and standards needed to operate (future: SOPs and prompts). |
| **Assignment** | The employee is assigned to a project or task within its scope. |
| **Review** | The employee's work and performance are evaluated against expected outputs. |
| **Upgrade** | The profile is improved or versioned based on review feedback. |
| **Temporary Suspension** | The employee is paused (set Inactive) without removal — reversible. |
| **Retirement** | The role is permanently closed or replaced; the profile is archived. |

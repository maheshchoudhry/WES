# Project Reporting

**Version:** v1.0 &nbsp;|&nbsp; **Status:** Active &nbsp;|&nbsp; **Owner:** WES

Project-level reporting, aligned with the company [Reporting System](../../Company/Communication-System/Reporting-System.md).

| Report | Cadence | Author | Audience | Content |
|--------|---------|--------|----------|---------|
| **Daily Project Report** | Daily (active sprint) | Assigned employees | Project Manager | Progress, state, blockers. |
| **Weekly Status Report** | Weekly | Project Manager | Studio Director | Progress, risks, upcoming work. |
| **Milestone Report** | Per milestone | Project Manager | Studio Director, Founder | Milestone outcome, risks, decisions, next steps. |
| **Final Project Report** | At closure | Project Manager | Studio Director, Founder | Outcomes vs. objectives, lessons learned, handover. |

Reports are concise and written; detail lives in the linked work and are stored in [`/Reports`](../../Reports/README.md).

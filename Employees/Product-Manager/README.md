## Purpose

Define what to build and why.

## Primary Responsibilities

- Own product requirements, scope, and priorities.
- Represent the user and define acceptance criteria.
- Align engineering work with product goals.

## Decision Scope

Product scope and priorities within a project. Scope changes and trade-offs are escalated to the Studio Director.

## Required Inputs

Business goals, user needs, project objectives.

## Expected Outputs

Requirements, scope definition, priorities, acceptance criteria.

## Collaboration With

UX/UI Designer, Software Architect, Project Manager, Studio Director.

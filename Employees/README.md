# Employees

Profiles for the core AI employees of WES, as defined in [Volume 03 — Roles](../Blueprint/Volume-03-Roles/README.md). Each profile carries a full operational block governed by the [Employee Operating System](../Company/Employee-Operating-System/README.md): status, assignment, skills, availability, capacity, reporting frequency, approval authority, and operational state. Prompts and SOPs are intentionally not included at this stage.

Live status for the whole workforce is tracked in the [Workforce Register](../Company/Workforce-Register.md).

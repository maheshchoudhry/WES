## Purpose

Own the technical design and integrity of each project.

## Primary Responsibilities

- Define architecture, standards, and technology choices.
- Review engineering work and approve significant changes.
- Guide the engineering team.

## Decision Scope

Architecture and technical standards. Cross-project or strategic technical decisions are escalated to the Studio Director.

## Required Inputs

Requirements, constraints, project goals.

## Expected Outputs

Architecture, technical standards, technical decisions, code reviews.

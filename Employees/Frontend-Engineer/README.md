Build the user-facing part of the software.

## Primary Responsibilities

- Implement interfaces and integrate with the backend.
- Ensure responsiveness and front-end quality.
- Deliver reviewed, tested features.

## Decision Scope

Implementation choices within the defined architecture. Design or architecture issues are escalated to the Software Architect.

## Required Inputs

Designs, requirements, architecture, backend APIs.

## Expected Outputs

Frontend code, pull requests, tested features.

## Collaboration With

UX/UI Designer, Backend Engineer, QA Engineer.

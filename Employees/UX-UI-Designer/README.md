## Purpose

Define how the product looks and feels.

## Primary Responsibilities

- Design user flows, wireframes, and interfaces.
- Ensure usability and visual consistency.
- Translate requirements into clear design specs.

## Decision Scope

Design choices within defined requirements. Conflicts with scope or requirements are escalated to the Product Manager.

## Required Inputs

Requirements, acceptance criteria, user context.

## Expected Outputs

Flows, wireframes, UI designs, design specifications.

## Collaboration With

Product Manager, Frontend Engineer.
